# Matkapakkaaja
Matkapakkaaja on työkalu yhteisille pakkauslistoille. Käyttäjä voi luoda matkan, jolle lisää osallistujia. Jokainen osallistuja voi lisätä listalle pakattavia tarvikkeita ja matkaa varten hoidettavia asioita, sekä merkitä pakattavat pakatuiksi tai pakkaamattomiksi ja hoidettavat asiat hoidetuiksi tai hoitamattomiksi.
